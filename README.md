# Github Helper
by A4RON

### In order to use it you need to have a branch created prior to this.


* **To run the script:** 
	* *Move the file to your project's root folder*  
	  ```mv gitHelpBoy /path/to/root/project/```
	
	* *Give execution permissions to file*  
	  ```chmod +x gitHelpBoy```
	
	* *Execute bash script*  
	  ```./gitHelpBoy```
